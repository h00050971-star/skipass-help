# Проект: skipas-krasnaya-polyana

## Что это

Лендинг-агрегатор для покупки скипасса на Красной Поляне. Три курорта — одна страница. Никаких продаж через нас, только прямые ссылки на официальные кассы курортов.

**Целевой URL:** будущий домен типа `skipas-krasnaya-polyana.ru`, хостинг — Vercel.

---

## Стек

- Чистый HTML + inline CSS (один файл, никаких фреймворков)
- Шрифт: Inter (Google Fonts)
- Фотографии: локальные `.webp` в `assets/`, фото Розы Хутор — внешний URL `https://s3.rosakhutor.ru/...` с `onerror` fallback
- Логотипы курортов: PNG в `assets/`, но в карточках сейчас используются **inline SVG-кружки + текст** (не PNG)

---

## Структура страницы

1. **Sticky header** — логотип сайта + навигация
2. **Page head** — H1 + лид
3. **Resort cards** (3 карточки) — фото + бейдж слева + лого-пилюля справа + высота + тело с описанием, статами и кнопкой
4. **Фильтр-виджет** — тип билета / дата / взрослые / дети
5. **Как купить онлайн** — 4 шага с иконками
6. **Сравнительная секция** — таблица по курортам
7. **SEO-статьи** — 6 карточек по темам (электричка, аэропорт, веб-камеры и т.д.)
8. **Футер**

## Лого-пилюли на карточках (правый верхний угол фото)

Стилизованы под левые бейджи (`.badge`) — одинаковая высота, шрифт, скругление. Внутри: цветной SVG-кружок + название курорта текстом.

```css
.resort .photo .resort-logo {
  position: absolute; top: 16px; right: 16px;
  background: rgba(255,255,255,0.95);
  border-radius: 999px;
  padding: 7px 12px;
  display: inline-flex;
  align-items: center;
  justify-content: center;
  gap: 5px;
  font-size: 11.5px;
  font-weight: 700;
  text-transform: uppercase;
  letter-spacing: 0.06em;
  color: var(--ink);
}
```

Цвета кружков: RK `#0C1A6B` (черничный), KP `#D81B21` (красный), GZ `#0079C2` (синий).
